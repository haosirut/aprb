## 📖 Описание

GateRunner — минималистичная 2D «бегалка» (vertical runner) в бесконечном режиме. Игрок управляет отрядом юнитов, проходя через ворота с математическими операциями (+, -, *, /) и сталкиваясь с врагами-блоками. Все игровые объекты движутся сверху вниз. Игрок зафиксирован в нижней части экрана и перемещается только по горизонтали. Игра бесконечная — конец только при потере всех юнитов.

## 🕹 Игровой функционал

### Жанр и механика
- **Vertical runner** в бесконечном режиме (infinite)
- Управление прямоугольником шириной ровно 1 полоса
- Все арифметические операции с юнитами используют `Math.floor()` и минимум `1`
- Скорость увеличивается на 3% после каждого врага (compound)

### Управление
- **Тач/мышь** — горизонтальное перемещение игрока
- Зона управления: нижние 15% экрана
- Позиция игрока: фиксированная по Y — `canvas.height * 0.95`
- Размер: `width = canvas.width / 5`, `height = width` (квадрат, 1 полоса)
- Клэмпинг: `clamp(x, 0, canvasWidth - playerWidth)`
- Линейная интерполяция (lerp) для плавного движения

### Игровые объекты

#### Игрок (`src/entities/Player.js`)
- Визуал: белый квадрат (1 полоса x 1 полоса) с обводкой
- Размер фиксирован: `width = canvas.width / 5`, `height = width`
- Вертикальная позиция: `y = canvas.height * 0.95`
- Вспышка (мерцание) при получении урона
- Пересчёт позиции при `resize` окна

#### Ворота (`src/entities/Gate.js`)
- Спавнятся **парами** по сегментному алгоритму:
  - **Первые ворота**: тип `−` или `/` (негативные), ширина `randomInt(1, 4)` полос
  - **Свободные полосы** разбиваются на непрерывные сегменты
  - **Вторые ворота**: тип `+` или `*` (позитивные), в случайном сегменте
  - Ворота НЕ обязаны заполнять все 5 полос
- 3 визуальные зоны: левый столб `#4a4a4a` (10px), зона эффекта, правый столб `#4a4a4a` (10px)

| Тип | Символ | Цвет зоны | Значение | Роль | Эффект |
|-----|--------|-----------|----------|------|--------|
| INCREASE | `+` | `#22c55e` | `randomInt(2, 100)` | Позитивное (2-е) | `units + value` |
| DECREASE | `−` | `#ef4444` | `randomInt(2, 100)` | Негативное (1-е) | `units − value` |
| MULTIPLY | `*` | `#3b82f6` | `randomInt(2, 10)` | Позитивное (2-е) | `units * value` |
| DIVIDE | `/` | `#f97316` | `randomInt(2, 10)` | Негативное (1-е) | `units / value` |

- Скорость: `BASE_SPEED * speedMultiplier` (150 * 1.03^n)
- `Math.max(1, Math.floor())` после каждой операции

#### Враги (`src/entities/Enemy.js`)
- Визуал: сетка блоков (кубиков), привязанных к полосам экрана
- **Ширина**: `randomInt(1, 5)` полос, стартовая полоса `randomInt(1, 6 - widthLanes)`
- **Размер кубика**: `cubeSize = lanePx / 5` (динамический)
- **Сетка**: `cols = widthLanes * 5`, `rows = 5`
- Вероятности рядов: `[1.0, 0.9, 0.8, 0.7, 0.6]`
- Гравитация: под пустотой не может быть кубика
- HP: секвенциальная формула `((Units OP1 V1) OP2 V2) * 2`
- HP на кубик: `Math.max(1, Math.floor(totalHP / activeSquares))`
- Цвет: fill `#b91c1c`, stroke `#7f1d1d`

#### Коллизия игрок ↔ враг
- AABB: прямоугольник игрока × квадрат врага
- При коллизии: `playerUnits -= Math.floor(square.hp)`

### Бесконечный режим
- Нет уровней, нет прогресс-бара, нет победы
- Спавн бесконечный: `[GATE_PAIR, GATE_PAIR, ENEMY]` → цикл
- Скорость: `speedMultiplier = Math.pow(1.03, enemySpawnCount)`
- Конец: `playerUnits <= 0` → экран поражения

### Интерфейс (HUD)
- Только HP-счётчик: чёрный прямоугольник 140×30, `HP: XX` (белый, monospace)
- Позиция: фиксированная (10px от верха, 10px от лева)
- Прогресс-бар удалён

### Состояния приложения
```
menu → playing → result → menu
```
| Состояние | Сцена | Описание |
|-----------|-------|----------|
| `menu` | MenuScene | Меню с кнопкой «ИГРАТЬ» |
| `playing` | GameScene | Бесконечная игра |
| `lost` | ResultScene | Экран поражения |

## 🔧 API

### State (`src/core/State.js`)
```
State.playerUnits               — текущее HP (минимум 1)
State.gameState                 — 'menu' | 'playing' | 'lost'
State.gateHistory               — [{type, value}] последних 2 ворот
calculateEnemyHP(units, history) — ((units OP1 v1) OP2 v2) * 2
```

### Spawner (`src/core/Spawner.js`)
```
new Spawner()                   — бесконечный спавнер
.update(dt, canvasWidth)        — возвращает Gate-инстансы / enemy-объекты
.speedMultiplier               — Math.pow(1.03, spawnCounter)
.spawnCounter                  — количество заспавненных врагов
```
Алгоритм: `[GATE_PAIR, GATE_PAIR, ENEMY]` → бесконечный цикл.
Скорость: `BASE_SPEED = 150px/s`, `speedMultiplier = 1.03^n`.

### Скорость
- `BASE_SPEED = 150px/s`
- `speedMultiplier = Math.pow(1.03, spawnCounter)` (compound +3%)
- Применяется ко всем движущимся объектам (ворота, враги)
